# streamline-client
* Front-end web client for streamline

## Node.js
* If you don't have node on your machine, follow this link and install : https://www.npmjs.com/get-npm

## Install angular CLI 
* To get the angular command line tool, run `npm install -g @angular/cli`

## Install project dependencies
* Go to the application root of the project folder streamline-client/streamline
* Run `npm install`

## Serve the application
* Run `ng serve` in the application root
* Visit localhost:8000/home
